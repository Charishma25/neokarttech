
# User Sign Up and Login System

1. Import 'users.sql' into your MySQL server to create the database and table.
2. Place all files into your MAMP 'htdocs' folder.
3. Start MAMP servers and visit 'localhost/signup.html' to register.
4. Use 'localhost/login.html' to log in.
5. Adjust 'config.php' database credentials if needed.
