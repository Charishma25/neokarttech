
# ✅ NeoKart User Authentication System

This project includes:
- Fully working PHP + MySQL backend for user sign up & login.
- NeoKart layout-matching forms (`signup.html` and `login.html`).
- Database SQL script (`users.sql`).
- `config.php` for DB connection (MAMP defaults: root/root).

## ⚙️ How to Run Locally
1. Install MAMP (Mac recommended).
2. Place these files in `Applications/MAMP/htdocs/neokarttech/`.
3. Start MAMP servers.
4. Go to `http://localhost:8888/neokarttech/signup.html` to sign up.
5. Then go to `login.html` to log in.
6. Use `phpMyAdmin` to import `users.sql` and create the `user_auth_db`.

## ⚡ Note
GitHub Pages does not run PHP — test this locally!
